# koa项目的cli工具

## 使用方法

1. 切换到某目录;
2. 执行`my-koa-init`命令;
3. 依次填入 项目名, 端口号, 中间件等;
4. 安装依赖并启动;

## 依赖包的功能

* `inquirer`: 交互式命令行库
* `ejs`: 渲染 (模板+数据) => 代码
* `execa`: 执行子进程
* `prettier`: 格式化代码
* `chalk`: 彩色输出
